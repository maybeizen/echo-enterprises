# Simply Smoother (Vulkan)

## About Vulkan

Vulkan is a Minecraft Mod that optimizes just about everything, by replacing Minecraft OpenGL renderer with a more optimized one, that runs on the Vulkan API

Despite being more optimized, Vulkan can be unstable and has little compatibility. This modpack has even less content than [Simply Smoother](https://modrinth.com/modpack/simplysmooth). **Use this modpack at your own risk, and do not expect stability.**

## Loaders

This Modpack is built upon Fabric, which is an experimental modding toolchain for Minecraft.

## Abilites/Tweaks

This modpack has all kinds of cool abilities and tweaks, here is just a small list:

- Quick Renders
- Improved Performance
- Tons of QOL
- Best for playing servers

## Minimal Mods

Keeping the Mod count low increases load times and performance, so we did that! At most, this modpack will contain 50 mods (Not including needed dependencies) and will consider the lower-spec hardware on older computers.

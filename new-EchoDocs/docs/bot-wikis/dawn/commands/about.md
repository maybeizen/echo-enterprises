# About

All about Dawn's commands and how they work

- Dawn uses **SLASH COMMANDS**, which are easy to use and are approved by Discord. This allows for faster command execution as well, as these can allow less code to run in the background to check for required permissions, arguments, etc.

- This also makes it easy for users (like You) to properly fill in the arguments and understand what the command does.

# Moderation

Here is a list of all moderation features that Dawn has:

**Primary Functions:**

- `ban` - Ban a user from the current guild.
- `kick` - Kick a user from the current guild.
- `timeout` - Mute/Timeout a user for a specific period.
- `untimeout` - Remove a timeout from a user.
- `channel` - Manage your current channel
- `role` - Configure roles for a user.

**Other Functions:**

- `autorole` - Configure the Autorole system.
- `members` - Show the member count of the current guild.

:::warning WARNING
Autorole will not work if you do not set a Welcome Channel! See the [**utility page**](utility) for more!
:::

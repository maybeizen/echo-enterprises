# Basic Info

**DAWN IS CURRENTLY IN REDEVELOPMENT AND IS NOT FULLY STABLE**

_Dawn_ is a multipurpose, high-functionality Discord Bot built with BDScript. It does all of the utility functions that you need it to do, from Moderation to even managing suggestions for your server!

Here are just a few things that _Dawn_ does:

- Welcome System 👋
- Moderation 🛡️
- Suggestions 💡
- Economy 🤑
- Support Tickets 🎫
- AutoRole 🚗
- User-Based Profiles 👤
- and MORE...

Interested in all of the epic things that Dawn does? Check out [this page](https://docs.echoenterprises.net/docs/bot-wikis/dawn/commands/about)
